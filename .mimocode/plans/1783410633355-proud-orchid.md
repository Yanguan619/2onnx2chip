# Plan: 项目改造为可 pip install 的 Python 包

## 目标
通过 `pip install git+<repo-url>` 安装后，所有脚本可通过命令行直接调用。

## 包名 & CLI 前缀
- Python 包名: `tonnx2chip`（import 用）
- CLI 命令前缀: `tonnx2chip-`

## 实施步骤

### Step 1: 创建 `pyproject.toml`
- build-system: setuptools
- 依赖: `typer[all]`, `onnx`, `numpy`, `Pillow`, `transformers`
- 可选依赖组:
  - `[infer]`: `ais_bench`
  - `[quant]`: `amct_onnx`
  - `[optimize]`: `onnxslim`
  - `[all]`: 以上全部
- `[project.scripts]` 定义 CLI 入口

### Step 2: 创建 `src/tonnx2chip/` 包骨架
- 所有子目录的 `__init__.py`
- `constants.py` 提取: `VOCAB_SIZE`, `HIDDEN_SIZE`, `TARGET_SEQ_LEN`, `EMBED_SEQ_LEN`, `FULL_ATTN_LAYERS`

### Step 3: 移动文件
- 12 个 .py 脚本 → 对应子目录（重命名去掉 `qwen3p5_` 前缀）
- 2 个 .sh → `export/` 和 `tools/`
- 2 个 config → `src/tonnx2chip/config/`
- 删除旧 `script/` 和 `config/`

### Step 4: 修复 import 路径
- `quant_nuq.py`: `from quant_uniform import ...` → `from tonnx2chip.quantize.quant_uniform import ...`
- `val_om.py`: `import memory_monitor` → `from tonnx2chip.tools.memory_monitor import MemoryMonitor`
- 配置文件: `Path(__file__).parent.parent / "config" / "nuq_base.cfg"`
- 常量: `from tonnx2chip.constants import VOCAB_SIZE, ...`

### Step 5: CLI 入口点
```
tonnx2chip-export       = tonnx2chip.export.export_onnx:app
tonnx2chip-optimize     = tonnx2chip.optimize.pad_to_slice:app
tonnx2chip-val-onnx     = tonnx2chip.validate.val_onnx:app
tonnx2chip-val-om       = tonnx2chip.validate.val_om:app
tonnx2chip-infer        = tonnx2chip.infer.infer_om:app
tonnx2chip-infer-fast   = tonnx2chip.infer.infer_om_fast:app
tonnx2chip-quant        = tonnx2chip.quantize.quant_uniform:app
tonnx2chip-quant-nuq    = tonnx2chip.quantize.quant_nuq:app
```

### Step 6: 更新 README.md
- 新目录结构
- 命令示例改为 `tonnx2chip-xxx` 形式
- 安装说明: `pip install git+<repo-url>`

### Step 7: 更新 .gitignore
- 添加 `src/tonnx2chip.egg-info/`, `dist/`, `build/`
